Tutorial github

Membuat repository

Klik new repository, add new file.

git clone

Copy url di clone or download ke dalam terminal (example : folder-tujuan/git clone https://github.com/danangrepo/tutorial.git untuk membuat folder baru tambahkan nama folder setelah url)

Melakukan perubahan code

setelah meng-clone repo, lakukan perubahan terhadap isi file. Lalu add seperti biasa (git add .) dan commit seperti biasa (git commit -m "message") setelah itu baru lakukan push ke github.

Git push dan Git pull

git push digunakan untuk mengupdate isi file dan git pull untuk menarik isi file ke local. Example :

  • git push origin master (ini untuk push ke branch master)
  • git pull

Branch dan Merge

Pertama buat dulu branch baru bisa menggunakan git branch nama-branch atau git checkout -b nama-branch

merge

Untuk melakukan merge bisa menggunakan git merge nama-branch-yang-ingin-digabung lalu git push origin master.

hapus branch

untuk menghapus branch bisa menggunakan git branch -D nama-branch kemudian untuk github ketik git push origin :nama-branch

Fork

digunakan untuk mengcopy repository orang lain ke repo kita

.gitignore (menggunakan (.))

digunakan untuk menyembunyikan file di github. cara penggunaan buat file dengan nama .gitignore dan ketik di dalam file tersebut nama file yang ingin disembunyikan kemudian git add . lalu commit dan push seperti biasa
